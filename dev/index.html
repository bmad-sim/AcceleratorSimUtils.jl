SimUtils.cos_one — Function
cos_one(x)

Function to calculate cos(x) - 1 to machine precision. This is usful if angle can be near zero where the direct evaluation of cos(x) - 1 is inaccurate.

SimUtils.gen_pinknoise — Function

function gen_pinknoise( beta::Float64=1.0, size::Int64=2^12, dt::Float64=1.0, f0::Float64=1.0 )

description:
> this function generates pink noise with frequency rolloff coefficient 'beta'

positional parameter:
beta             -- type:Float64, frequency rolloff coefficient
size             -- type:Int64, number of points
keyword parameter:
dt               -- type:Float64, the difference of between each data, total time = size * dt, unit: s
f0               -- type:Float64, the spectrum has the shape (f0/f)^(beta), unit: Hz

Note: 
the algorithm is based on 
"Timmer, J. and Koenig, M.: On generating power law noise. Astron. Astrophys. 300, 707-710 (1995)"

SimUtils.modulo2 — Function
function modulo2(x, amp)

Function to return

     mod2 = x + 2 * n * amp

where n is an integer chosen such that

    -amp <= mod2 < amp
